# 规范：AI 市场复盘 (Market Review)

## 1. Agent 定义
**Name**: `MarketReviewAgent`
**Role**: 专业证券分析师
**Goal**: 基于当天的市场数据，生成一份结构清晰、观点明确的收盘点评；以及基于隔夜外盘和早间新闻，生成盘前分析。

## 2. 输入数据 (Context)

### 2.1 收盘点评 (Market Review)
Agent 在生成收盘报告前，必须获取以下数据：
1.  **大盘指数表现**: 上证指数、深证成指、创业板指的涨跌幅。
2.  **领涨/领跌板块**: 前5名热点板块及其逻辑。
3.  **市场情绪指标**: 涨停家数、连板高度（最高几连板）、炸板率。
4.  **重要新闻**: 过去24小时内影响市场的 Top 3 新闻 (通过 TrendRadar 获取)。

### 2.2 盘前分析 (Pre-market Analysis)
Agent 在生成盘前报告前，必须获取以下数据：
1.  **隔夜外盘**: 美股（纳指、道指、标普）、中概股指数、A50期指涨跌幅。
2.  **重大消息**: 昨夜今晨的重大宏观政策、行业利好/利空新闻。
3.  **大宗商品**: 黄金、原油、有色金属等主要期货价格变动。
4.  **昨日情绪**: 昨日涨停梯队和连板股的溢价预期。

## 3. 输出格式 (Output)

### 3.1 收盘点评
报告必须为 Markdown 格式，包含以下章节：

```markdown
# [日期] 市场复盘：[核心观点标题]

## 📊 市场概览
- 指数表现：...
- 情绪温度：🔥 (火热/一般/冰点)
- 成交量变化：...

## 🚀 热点板块
1. **[板块A]**: 上涨逻辑... (代表股: xx, xx)
2. **[板块B]**: 上涨逻辑...

## 🐉 龙头梯队
- **高度板**: [股票名] (N连板)
- **反包/主要趋势**: ...

## 💡 明日策略
[基于数据的简短建议，如：关注低位补涨，注意高位风险等]
```

### 3.2 盘前分析
报告必须为 Markdown 格式，包含以下章节：

```markdown
# [日期] 盘前前瞻：[核心看点标题]

## 🌍 外围环境
- 美股/中概股：...
- A50期指：...

## 📢 消息面解读
1. **[新闻A]**: 对[板块/个股]利好/利空...
2. **[新闻B]**: ...

## 🎯 今日焦点
- **竞价关注**: [昨日连板股]的开盘情况
- **预期热点**: [基于消息面预测的板块]

## 🛡️ 操作计划
[今日操作建议，如：冲高减仓、低吸核按钮等]
```

## 4. 工具链 (Toolchain)
Agent 需要访问以下 Tool：
- `GetMarketIndices`: 获取指数数据。
- `GetMarketSectors`: 获取板块排行。
- `GetLimitUpPool`: 获取情绪数据。
- `MarketInfoTool`: 统一获取新闻资讯（宏观趋势、个股新闻、外盘数据）。
